# QR Code Generator

A full-stack QR Code Generator that allows users to create customizable QR codes for any URL.

## Features

- Generate QR codes for any URL
- Customize QR size
- Change foreground and background colors
- Download QR as PNG
- View recent QR generation history
- Delete QR history records

## Tech Stack

Frontend
- HTML
- CSS
- JavaScript

Backend
- Node.js
- Express.js

Database
- MySQL
